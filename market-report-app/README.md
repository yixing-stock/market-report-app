# Market Report App

A GitHub-based Claude Code project for generating dashboard-style stock market reports.

## What this project does

This repo is designed to help Claude Code produce structured market reports that feel like a dashboard instead of a chat reply.

The framework includes:
- multi-horizon analysis
- multi-agent research
- bull vs bear debate
- confidence breakdown
- narrative change detection
- scenario analysis
- dashboard-style HTML output
